# 📝 To-Do List Manager (CLI)

A simple **command-line To‑Do List Manager** built with **Node.js** using the `readline` module.  
Developed as part of Sprint 01 in the 3LOGY Software Development Bootcamp (2026).

---

## 🚀 Features
- Add Tasks: Create new tasks with a title, deadline, and category.
- View Tasks: Display all tasks with their details.
- Mark Complete: Update tasks as completed.
- Filter by Category: Show tasks belonging to a specific category (Work, Personal, Urgent).
- Exit Program: Gracefully close the application.

---

## 📂 Project Structure
Single file: `todo.js` (contains both README and code)

---

## ⚙️ Setup & Usage
1. Ensure you have **Node.js** installed (v16+ recommended).
2. Save this file as `todo.js`.
3. Run the program in your terminal:
   ```bash
   node todo.js
